# AI Assignments

This repository contains implementations of several Artificial Intelligence concepts including intelligent agents, search algorithms, and human verification systems.

---

# 1. AQI Intelligent Agent

## Description
This project implements an **Air Quality Index (AQI) agent** that categorizes air quality levels based on AQI values. The agent reads AQI data and determines the pollution category.

## Components
- `code.py` – Main agent implementation
- `config.py` – Configuration settings
- `aqi_categories.csv` – AQI category definitions
- `aqi_checkpoints.csv` – AQI dataset for testing

## Agent Architecture
The AQI system follows a **Simple Reflex Agent architecture**:
1. Sensor reads AQI values.
2. Agent checks category rules.
3. Actuator outputs the air quality category.

# 2. Turing Test & CAPTCHA

## Turing Test

### Description
A simulation of the **Turing Test** where a human judge attempts to distinguish between responses from a machine and a human.

### Implementation
- `turing.py`

### Architecture
Components:
1. Judge (user input)
2. Human responder
3. Machine responder
4. Evaluation module

The system randomly presents responses and the judge must guess which one is generated by the machine.

# 3. Water Jug Problem (Search Algorithms)

## Description
Implementation of **Uninformed Search Algorithms** to solve the classic **Water Jug Problem**.

Jugs:
- Jug 1 = 4 liters
- Jug 2 = 3 liters

Goal:
Measure exactly **2 liters** of water.

---

## Algorithms Implemented

### Breadth First Search (BFS)
- Explores level by level
- Guarantees optimal solution

### Depth First Search (DFS)
- Explores deep paths first
- Uses less memory but not always optimal

---

## Files

- `water_jug.py` – Problem definition
- `bfs.py` – BFS implementation
- `dfs.py` – DFS implementation
- `main.py` – Runs the search algorithms

---

## Performance Comparison

| Algorithm | Complete | Optimal | Memory |
|----------|----------|--------|--------|
| BFS | Yes | Yes | High |
| DFS | Yes | No | Low |
